# pesquisa-eleitoral-df
Pesquisa Eleitoral DF 2026 - Sistema de simulação de pesquisas eleitorais utilizando agentes de IA que representam 400+ perfis sintéticos de eleitores do Distrito Federal. Cada agente possui características demográficas, socioeconômicas, políticas e psicológicas realistas, permitindo simular entrevistas e analisar tendências eleitorais.
