## Como Usar

### Para a IA
Ao implementar uma nova feature, a IA deve:
1. Identificar o tipo de código a ser criado
2. Ler o exemplo correspondente
3. Seguir o padrão estabelecido
4. Adaptar para o caso específico

### Para Desenvolvedores
- Manter exemplos atualizados com as melhores práticas
- Adicionar novos exemplos quando padrões emergirem
- Documentar mudanças significativas

## Convenções

### Python (Backend)
- Type hints obrigatórios
- Docstrings em funções públicas
- Logging estruturado
- Tratamento de erros explícito

### TypeScript (Frontend)
- Interface de Props sempre definida
- Componentes funcionais
- Hooks customizados para lógica
- Tailwind para estilização
