# Pesquisas Premium (Pacotes Auditáveis)

Esta pasta guarda as execuções de pesquisa no padrão **INTEIA Premium**.

Cada pesquisa deve criar um subdiretório no formato:

`{slug}_{YYYYMMDD_HHMM}/`

Conteúdo mínimo por pesquisa:

- `README.md` (índice do pacote)
- `CHECKLIST.md`
- `PLANO_PESQUISA_COMPLETO.md`
- `QUESTIONARIO.md`
- `DADOS_BRUTOS.json`
- `VALIDACAO.md`
- `INSIGHTS.md`
- `PREDICOES.md`
- `RELATORIO_CLIENTE.md`
